# FitDoggy

An exercise, food, and weight tracker for your pups.

See it in action [here](https://fitdoggy.herokuapp.com/).

## Getting Started
1. Clone the repo
2. Install required packages
```
npm install
```
3. Start the server
```
python server.py
```
4. Visit the [app](http://localhost:5000/)

### Prerequisites

```
Python 2.7.10
Flask 0.12.2
Flask_SQLAlchemy 2.2
node v4.5.0
npm 2.15.9
```
